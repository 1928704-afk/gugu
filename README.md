# 고구마 전도

복음을 전하며 고구마를 키우는 게임입니다.

## 실행 방법

1. 의존성 설치
```bash
npm install
```

2. 서버 실행
```bash
npm start
```

3. 브라우저에서 접속
- http://localhost:3001

## 기능

- **시작하기** → 자신의 이름 입력 후 게임 시작
- 유저별 고구마 이름·레벨(HP)은 **데이터베이스(SQLite)**에 저장
- **세션** 사용 → 같은 브라우저로 재접속 시 이름 입력 없이 이어서 플레이 가능
- **미션 보상**
  - 일일 미션: `말씀 + 기도 + 연락` 완료 시 전체 고구마 온도 `+4`
  - 주간 미션: 같은 주(월~일) `활동 5일` 달성 시 전체 고구마 온도 `+8`
- **성장 히스토리**
  - 고구마 상세 화면에서 최근 7일 행동 점수 그래프 확인
  - 최근 행동 로그(행동 종류/날짜) 확인
  - 최근 7일 총점/평균/최고 성장일, 행동 비율(상위 3개) 통계 제공
- **전직 패시브(심화)**
  - 고구마의 주 행동(전직)과 같은 행동을 할 때 추가 보너스 `+1` 적용
- **행동 쿨다운 가시화**
  - 오늘 사용한 행동 버튼은 `오늘 완료` 상태로 비활성화
  - 다음 행동 리셋까지 남은 시간(KST 기준) 카운트다운 표시
- **주간 미션 로테이션**
  - 주간 미션이 유저/주차 기준으로 로테이션되어 매주 다른 목표가 제시됨
  - 예: 주간 출석 5일, 만남 2회, 게시판 작성 3회

## 데이터
